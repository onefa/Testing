1. В фоновом режиме приложение записывает в локальную БД прогноз по последним 7 городам.
Можно сделать и больше, конечно.
2. Возможен оффлайн доступ к истории запросов (кнопка Free).
3. При подключении нового пользователя первый запрос прогноза производится с
автоматическим определением города по геолокации.
4. Ввиду того, что при работающем приложении список результатов запросов может достигать
значительной длины, реализована автопрокрутка на текущую дату/время.
